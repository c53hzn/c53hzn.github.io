Vue开发的注意事项

前言

今天给我的网站的总博客列表页、特定标签博客列表页、特定语言博客列表页新增了侧边栏，用来显示年份索引。

年份索引在桌面版时显示在右侧，手机版时隐藏，可使用固定在页面右下角的列表按钮进行开关。

就这么个功能做了一整天。不过做的时候遇到一些问题，又学到了一些关于 Vue.js 开发的小知识。

（并没有，只是没看文档就瞎写的后遗症而已）

应用组件时无法内嵌Vue表达式

要在组件里内嵌 Vue 表达式或者 HTML 的话，需要在原组件中把内嵌的位置写上 <slot></slot> ，这就是在应用组件时插入表达式或 HTML 的插槽。

要注意的是，插槽不可以是根元素标签，必须写在内嵌位置的里面。

组件内写法如下：

<div class="side-nav">
  <slot></slot>
</div>

应用该组件的上一级元素里这样写：

<div>
  <h2>This is a title</h2>
  <SideNav class="side-unit" :isShow="isSideNavShow">
    <ul class="year-list">
      <li v-for="(year,i) in years" :key="i">
        {{year}}
      </li>
    </ul>
  </SideNav>
</div>

最终渲染的结果如下：

<div>
  <h2>This is a title</h2>
  <div class="side-nav side-unit">
    <ul class="year-list">
      <li>
        2021(18)
      </li>
      <li>
        2020(10)
      </li>
    </ul>
  </div>
</div>

应用组件时无法绑定点击事件

本来如果要在 HTML 标签上绑定点击事件的时候只需要 v-on:click="xxx" 或者 @click="xxx" 就可以了，但是如果是组件的话就需要这么写

<ListButton @click.native="toggleSideNav"></ListButton>

无法传递布尔值给组件

本来如果要传递固定的字符串给组件的 props 时不需要用 v-bind: 或者 : 符号，但是如果要传递的是布尔值，就需要了。
