How to use:

  1. Download the zip file
  2. unzip the file to folder
  3. click "app.html" to open in your browser (Latest version of Chrome is recommended)
  4. Input your shop info and item info in the editing area
  5. Click "generate HTML" button
  6. Preview result, you may switch between desktop version and mobile version
  7. Click "Copy HTML" button to copy content
  8. Log in to eBay, go to your item editing page, scroll down to description section, turn to HTML code mode, paste the content there

Points to notice:

  1. Please only add links within eBay
  2. Please only use photo links that end with a valid extensions like "jpg" or "png"
  3. Please do not use any links that are not starting with "https"
